# What is TPaaS Email Service?

Email Service is one of the micro services provided by TPaaS that provides a common platform for multiple Trimble Applications across various divisions, to send emails to their end users.  Integration with TpaaS email service is a simple process and using it eliminates the need to build, and maintain an email system in-house, allowing you to focus on other facets of product development. 

### Features

* Create and manage email template in HTML format to send emails
* Ability to inject dynamic contents while sending emails using template
* Supports templates with multiple languages
* Register your own domain to send emails to your customers
* Ability to send raw messages
* Know data relating to bounced emails, rejected emails, sent emails
* Get your application's Email Usage stats
